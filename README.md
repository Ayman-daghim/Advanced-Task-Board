# Advanced-Task-Board
Advanced task board UI built with pure HTML and CSS.

## Features
- Responsive two-column layout
- Task board and notes board
- Achievement table section
- Floating action button
- Modern gradient backgrounds
- Clean UI design
- RTL Arabic layout support

## Technologies Used
- HTML5
- CSS3

## Concepts Practiced
This project helped practice:

- Float and clear
- Positioning
- Fixed buttons
- Linear gradients
- CSS grouping
- Tables styling
- Responsive sizing using `calc()`
